# Programmation3D-Step-1

Pour déplacer le ball (Sphere) 

- Z ==> Avant
- S ==> Arrière
- D ==> Droit
- Q ==> Gauche

- H ==> Haut
- B ==> Bas


- P ==> Play Avant 
- M ==> Play Arrière

==> À chaque fois le ballon passe le mur en face, la couleur s'échange avec la fonction Math.random().
